# 📝 Character Counter

This project is a responsive **Character Counter** built using HTML, CSS, and JavaScript. It tracks user input in real time, displays the number of characters typed, shows the remaining characters, and prevents users from exceeding the maximum limit of **200 characters**.

---

## 🚀 Features

✅ Real-time character counting
✅ Displays typed character count
✅ Shows remaining characters
✅ Maximum limit of 200 characters
✅ Prevents input beyond the character limit
✅ Warning message when the limit is reached
✅ Responsive design for desktop and mobile devices
✅ Clean and beginner-friendly user interface

## 🎯 Functionality

### ✍️ Text Input

* Users can type a message in the text area.
* Maximum of **200 characters** allowed.

### 🔢 Character Counter

* Displays the number of characters typed.
* Displays the remaining characters available.
* Updates automatically as the user types.

### ⚠️ Character Limit

* Prevents users from typing more than **200 characters**.
* Displays a warning message when the limit is reached.
